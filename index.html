jQuery.liveFilter

About

A jQuery plugin to filter a list of data or elements. A fork of Mike Merritt's jQuery.liveFilter plugin.

Code

HTML:

<input id="livefilter-input" type="text" value="">
<ul id="livefilter-list">
  <li><a href="#">cool</a></li>
  <li><a href="#">nice</a></li>
  <li><a href="#">interesting</a></li>
  <li><a href="#">javascript</a></li>
  <li><a href="#">css</a></li>

  <li><a href="#">international</a></li>
</ul>

JavaScript:

$('#livefilter-list').liveFilter('#livefilter-input', 'li', {
  filterChildSelector: 'a'
});

Documentation

Syntax:

$(container element of elements to be filtered).liveFilter(input element, element to be filtered, options);

Options:

  • filterChildSelector - (string, optional) A string indicating the selector of the elements to be matched and filtered.
  • before - (function, optional) Executed before data or elements are filtered.
  • after - (function, optional) Executed after data or elements are filtered.
